# TrendPulse Daily

![App Preview](https://imgix.cosmicjs.com/5fbd6e40-d616-11f0-b30e-1d604711707a-photo-1556761175-b413da4baf72-1765405203085.jpg?w=1200&h=300&fit=crop,compress&auto=format)

A modern AI-powered autonomous news publication delivering technology, business, and AI innovation news. Built with Next.js 16, TypeScript, Tailwind CSS, and powered by Cosmic CMS.

## ‚ú® Features

- **Modern Tech News Design**: Clean, professional aesthetic optimized for readability
- **Dark Mode Support**: Seamless dark/light mode toggle with persistent preferences
- **AI-Generated Content**: Showcase AI-powered articles with model attribution and transparency
- **Trending Topics**: Real-time trending scores with visual indicators for hot content
- **Smart Content Discovery**: Category filtering, tag navigation, and advanced search
- **Weekly Roundups**: Curated weekly summaries with top articles and statistics
- **Newsletter Integration**: Built-in subscription system with email validation
- **Performance Optimized**: ISR with 1-hour revalidation, optimized images, skeleton loaders
- **SEO Excellence**: Dynamic metadata, Open Graph tags, JSON-LD structured data, automatic sitemap
- **Responsive Design**: Mobile-first approach with perfect layouts across all devices
- **Interactive Features**: Reading progress bars, view tracking, social sharing, table of contents

## üöÄ Getting Started

### Prerequisites

- Node.js 18+ or Bun runtime
- A Cosmic account with the TrendPulse Daily content model
- Vercel account (for deployment)

### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd trendpulse-daily
```

2. **Install dependencies**
```bash
bun install
```

3. **Set up environment variables**

Create a `.env.local` file in the root directory:

```env
COSMIC_BUCKET_SLUG=your-bucket-slug
COSMIC_READ_KEY=your-read-key
COSMIC_WRITE_KEY=your-write-key
RESEND_API_KEY=your-resend-api-key-optional
```

4. **Run the development server**
```bash
bun run dev
```

5. **Open your browser**

Navigate to [http://localhost:3000](http://localhost:3000)

## üìö Cosmic SDK Examples

### Fetching Articles with Categories and Tags

```typescript
import { cosmic } from '@/lib/cosmic'

// Get all published articles with nested categories and tags
const { objects: articles } = await cosmic.objects
  .find({
    type: 'articles',
    'metadata.status.key': 'published'
  })
  .props(['id', 'title', 'slug', 'metadata'])
  .depth(1) // Fetches nested category and tag data

// Sort manually by publish date (newest first)
const sortedArticles = articles.sort((a, b) => {
  const dateA = new Date(a.metadata.publish_date).getTime()
  const dateB = new Date(b.metadata.publish_date).getTime()
  return dateB - dateA
})
```

### Increment Article View Count

```typescript
// Update view count for an article
await cosmic.objects.updateOne(articleId, {
  metadata: {
    view_count: currentViews + 1
  }
})
```

### Subscribe to Newsletter

```typescript
// Create a new newsletter subscriber
await cosmic.objects.insertOne({
  type: 'newsletter-subscribers',
  title: email,
  metadata: {
    email: email,
    subscribed_at: new Date().toISOString(),
    active: true
  }
})
```

## üé® Cosmic CMS Integration

This application integrates seamlessly with Cosmic CMS using the following content types:

- **Articles**: Main content with rich text, categories, tags, and trending scores
- **Categories**: Article categorization with SEO metadata
- **Tags**: Content tags for discovery
- **Weekly Roundups**: Curated weekly summaries
- **Trending Topics**: AI agent research output (future feature)
- **Newsletter Subscribers**: Email subscription management

All content is fetched server-side for optimal performance and SEO, with ISR revalidation every hour to keep content fresh.

## üö¢ Deployment Options

### Deploy to Vercel (Recommended)

1. **Push your code to GitHub**

2. **Import your repository in Vercel**
   - Visit [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository

3. **Configure environment variables**
   - Add your Cosmic credentials in the Vercel dashboard
   - `COSMIC_BUCKET_SLUG`
   - `COSMIC_READ_KEY`
   - `COSMIC_WRITE_KEY`
   - `RESEND_API_KEY` (optional)

4. **Deploy**
   - Vercel will automatically build and deploy your site
   - Your site will be live at `https://your-project.vercel.app`

### Alternative Deployment

You can also deploy to:
- **Netlify**: Configure build command as `bun run build` and publish directory as `.next`
